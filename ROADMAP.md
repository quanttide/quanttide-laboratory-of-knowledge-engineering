## v0.5.0+（规划中）

### 多源联合

支持文档 + 数据库表 + API 响应多源交叉建模。

- 数据源抽象接口（FileSource, DBSource, APISource）
- 联合查询与跨源关联
- 统一三元组融合策略

### 增量抽取

源文档变更后仅重抽取差异内容。

- 文件哈希缓存（SHA256 → 跳过未变更文件）
- Git-aware 差异检测（仅处理 diff 文件）
- 增量合并到已有三元组

### 自动规则挖掘

从频繁子图生成 Datalog 规则候选。

- 子图模式挖掘（Apriori / gSpan）
- 规则置信度评估
- 人工确认工作流

### 可视化看板

候选事实人工确认界面。

- Web UI（React / Vue）
- 三元组浏览器（按置信度 / 来源 / 谓词过滤）
- 批量确认 / 拒绝 / 修改

### CI 流水线

知识库变更触发自动回归推理。

- GitHub Actions：push 时运行完整 pipeline
- 推理结果快照对比（diff 检测）
- 质量门禁（置信度分布、规则覆盖率）
