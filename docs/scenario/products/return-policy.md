# 退换货政策

## 退货条件

顾客在收到商品后 30 天内可以申请退货。退换货需要商品保持原包装且不影响二次销售。

## 退款流程

1. 提交退货申请
2. 客服审核（1-2 个工作日）
3. 寄回商品并填写物流单号
4. 仓库验收（3-5 个工作日）
5. 退款原路返回（7-15 个工作日）

## 例外情况

以下情况不支持退换货：
- 已激活的软件和数字商品
- 超过退货期限的商品
- 外观有明显人为损坏的商品

退换货期限是 30 天。
